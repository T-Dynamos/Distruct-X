# Installation
Clone Repo
```
git clone https://github.com/T-Dynamos/Distruct-X
```
Initialise Dir
```
mkdir ~/.android
```
Change Dir
```
cd Distruct-X
```
Run tool
```
pyhon3 distruct.py
```
